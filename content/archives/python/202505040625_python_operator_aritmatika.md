```python
a = 5
b = 2

print(a + b)  # Penjumlahan
print(a - b)  # Pengurangan
print(a * b)  # Perkalian
print(a / b)  # Pembagian
print(a // b) # Pembagian bulat
print(a % b)  # Sisa bagi
print(a ** b) # Pangkat
```
